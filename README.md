# 📰 NewsSphere - Live News Feed

A simple and interactive **Live News Feed** web application built using **HTML, CSS, and JavaScript**. The application fetches real-time news articles using the **GNews API** and allows users to search news by keyword, browse different categories, and switch between light and dark themes.

---

## 📌 Project Overview

This project was developed as part of our **On Job Training (OJT) End Semester Capstone Project**. The main objective was to build a JavaScript-focused web application by applying concepts such as DOM Manipulation, Event Handling, Fetch API, Async/Await, and Error Handling.

---

## ✨ Features

* 🔍 Search news using keywords
* 📰 View live news articles
* 📂 Browse news by categories

  * Technology
  * Business
  * Sports
  * Health
* 🌙 Light / Dark Theme Toggle
* ⚡ Loading message while fetching news
* ❌ User-friendly error handling
* 📱 Responsive layout

## 🚀 How to Run

1. Clone this repository.

```
git clone https://github.com/abhishek-d19/Live-News-Feed-.git
```

2. Open the project folder.

3. Open `index.html` using **Live Server** or any modern web browser.

4. Enjoy exploring live news!
